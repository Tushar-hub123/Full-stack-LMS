📚 Library Management System (MERN Stack)

A full-stack Library Management System built using the MERN stack that helps manage books, users, and borrowing activities digitally.
The application allows administrators to organize the library database while enabling users to search, borrow, and return books through an intuitive web interface.

🌐 Live Demo

Backend API (Render)
https://lms-yce2.onrender.com/api

✨ Key Features

Secure user authentication (Signup / Login)

Add, edit, or remove books from the catalog

Search functionality for books by title, author, or category

Issue and return book management

Monitor available and borrowed books

Manage library members and records

Clean and responsive user interface

REST API integration for frontend–backend communication
