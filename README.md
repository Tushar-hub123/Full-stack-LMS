# 📚 Library Management System (Full Stack)

A Full Stack Library Management System that allows administrators to manage books and users while enabling students to search, borrow, and return books efficiently.  
The system simplifies library operations by digitizing book records, user management, and borrowing processes through a modern web interface.

---

## 🚀 Features

- User Authentication (Login / Register)
- Add, update, and delete books
- Search books by title, author, or category
- Issue and return books
- Track available and borrowed books
- Manage library users
- Responsive and user-friendly interface
- RESTful API for backend operations
